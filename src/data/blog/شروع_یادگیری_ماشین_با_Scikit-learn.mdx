**Scikit-learn** یکی از محبوب‌ترین کتابخانه‌های پایتون برای یادگیری ماشین است و ابزارهای متنوعی برای آموزش، ارزیابی و استفاده از مدل‌های مختلف در اختیار توسعه‌دهندگان قرار می‌دهد.

## چرا Scikit-learn؟

- یادگیری آسان
- مستندات کامل
- الگوریتم‌های متنوع
- مناسب برای پروژه‌های واقعی

## نصب

```bash
pip install scikit-learn
```

## اولین مدل

```python
from sklearn.linear_model import LinearRegression

model = LinearRegression()
model.fit(X_train, y_train)
```

## الگوریتم‌های پرکاربرد

- Linear Regression
- Logistic Regression
- Decision Tree
- Random Forest
- KNN
- Support Vector Machine

## ارزیابی مدل

قبل از استفاده از مدل در پروژه، عملکرد آن را با معیارهایی مانند Accuracy، Precision، Recall یا RMSE بررسی کنید.

## جمع‌بندی

اگر قصد ورود به دنیای هوش مصنوعی و یادگیری ماشین را دارید، Scikit-learn بهترین نقطه شروع است و مفاهیم پایه را به‌خوبی آموزش می‌دهد.
