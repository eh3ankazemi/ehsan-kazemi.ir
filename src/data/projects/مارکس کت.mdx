# معرفی

**مارکس کت** یک پلتفرم بازار ارزهای دیجیتال است که با هدف نمایش اطلاعات لحظه‌ای بازار، نمودارهای تحلیلی و ارائه تجربه‌ای سریع و روان برای کاربران توسعه داده شده است. این پروژه با تمرکز بر عملکرد، رابط کاربری مدرن و پشتیبانی از چند زبان طراحی شده است.

## امکانات

- **نمایش لحظه‌ای اطلاعات بازار** ارزهای دیجیتال.
- **نمودارهای تعاملی** برای بررسی تغییرات قیمت.
- **جستجو و فیلتر** سریع ارزهای دیجیتال.
- **پشتیبانی از چند زبان (i18n)**.
- **قابلیت نصب به‌عنوان PWA** روی موبایل و دسکتاپ.
- **طراحی واکنش‌گرا** برای تمامی دستگاه‌ها.

## جزئیات فنی

- **فریمورک:** Next.js و React
- **زبان برنامه‌نویسی:** TypeScript
- **استایل‌دهی:** Tailwind CSS و SCSS
- **نمایش نمودارها:** Chart.js
- **بک‌اند:** Next.js API Routes
- **پایگاه داده:** MongoDB
- **چندزبانه:** i18n
- **استقرار:** Vercel

## چالش‌ها

یکی از مهم‌ترین چالش‌های پروژه، مدیریت اطلاعات بازار و نمایش سریع داده‌ها بدون کاهش عملکرد بود. برای حل این مشکل، درخواست‌های API بهینه‌سازی شدند، داده‌های موردنیاز کش شدند و کامپوننت‌ها به گونه‌ای طراحی شدند که فقط در زمان لازم دوباره رندر شوند.

## نتیجه

این پروژه توانایی من در توسعه اپلیکیشن‌های فول‌استک مدرن را به نمایش می‌گذارد. استفاده از فناوری‌های به‌روز، طراحی رابط کاربری واکنش‌گرا، پشتیبانی از چند زبان و نمایش داده‌های مالی به‌صورت لحظه‌ای، این پروژه را به نمونه‌ای کاربردی از توسعه وب مدرن تبدیل کرده است.
