# معرفی

تاریخ و زمان فارسی یک وب‌اپلیکیشن ساده و سبک است که تاریخ و ساعت فعلی را به‌صورت لحظه‌ای نمایش می‌دهد. این پروژه از API داخلی `Intl.DateTimeFormat` جاوااسکریپت برای نمایش تاریخ شمسی و میلادی استفاده می‌کند و بدون نیاز به کتابخانه‌های جانبی، اطلاعات را با دقت بالا نمایش می‌دهد.

## امکانات

- نمایش ساعت به‌صورت لحظه‌ای
- نمایش تاریخ شمسی و میلادی
- پشتیبانی از زبان فارسی و انگلیسی
- طراحی واکنش‌گرا
- بدون وابستگی به کتابخانه‌های تاریخ

## جزئیات فنی

- TypeScript
- HTML
- Vite
- Intl.DateTimeFormat API

## نتیجه

این پروژه نمونه‌ای ساده اما کاربردی از استفاده از APIهای مرورگر برای مدیریت تاریخ، زمان و بومی‌سازی در برنامه‌های تحت وب است.
