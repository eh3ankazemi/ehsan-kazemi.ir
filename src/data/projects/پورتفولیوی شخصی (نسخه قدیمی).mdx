# معرفی

این پروژه نسخه قبلی وب‌سایت شخصی من بود که با Next.js و Tailwind CSS توسعه داده شد. هدف آن معرفی پروژه‌ها، مهارت‌ها و تجربیات من بود و نقش مهمی در یادگیری معماری پروژه‌های مدرن و طراحی رابط کاربری داشت.

اگرچه این پروژه دیگر توسعه داده نمی‌شود، اما پایه و الهام‌بخش نسخه جدید وب‌سایت شخصی من بوده است.

## امکانات

- طراحی واکنش‌گرا
- معرفی پروژه‌ها و مهارت‌ها
- رابط کاربری مدرن
- عملکرد سریع با Next.js

## جزئیات فنی

- Next.js
- TypeScript
- React
- Tailwind CSS

## نتیجه

توسعه این پروژه تجربه ارزشمندی در زمینه معماری پروژه، طراحی رابط کاربری و توسعه وب مدرن برای من ایجاد کرد و بسیاری از ایده‌های آن در نسخه جدید سایت شخصی‌ام بهبود یافتند.
