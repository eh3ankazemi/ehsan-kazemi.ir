# معرفی

**لینک‌های شخصی** یک صفحه «لینک در بیو» (Link in Bio) است که برای مدیریت و اشتراک‌گذاری تمامی لینک‌های مهم من، از جمله وب‌سایت شخصی، شبکه‌های اجتماعی، پروژه‌ها و مخازن GitHub طراحی شده است. این پروژه به‌صورت کاملاً شخصی میزبانی می‌شود و وابسته به سرویس‌های شخص ثالث نیست.

## امکانات

- نمایش تمامی لینک‌های مهم در یک صفحه
- طراحی ساده، مدرن و واکنش‌گرا
- سرعت بارگذاری بالا و حجم کم
- پشتیبانی از دامنه اختصاصی
- امکان شخصی‌سازی آسان
- استقرار خودکار با GitHub Actions

## جزئیات فنی

- **زبان برنامه‌نویسی:** TypeScript
- **ابزار ساخت:** Vite
- **استایل‌دهی:** Tailwind CSS
- **میزبانی:** GitHub Pages
- **استقرار خودکار:** GitHub Actions

## چالش‌ها

هدف اصلی پروژه، ساخت یک صفحه سبک و سریع بدون استفاده از فریم‌ورک‌های فرانت‌اند بود. با استفاده از TypeScript، Tailwind CSS و قابلیت‌های مرورگر، پروژه‌ای ساده، قابل نگهداری و با عملکرد بالا توسعه داده شد.

## نتیجه

این پروژه راهکاری ساده و کاربردی برای مدیریت لینک‌های شخصی فراهم می‌کند و در عین حال توانایی من در توسعه وب‌سایت‌های سریع، سبک و بهینه با استفاده از فناوری‌های مدرن وب را به نمایش می‌گذارد.
