# Introduction

This was my previous personal portfolio website, built with **Next.js** and **Tailwind CSS**. It served as a place to showcase my projects, skills, and experiences while helping me learn modern web development practices.

Although I no longer maintain this project, it played an important role in my growth as a developer and inspired the portfolio you're viewing today.

## Features

- **Responsive Design** for desktop and mobile devices.
- **Project Showcase** with detailed information about my work.
- **About Me** section introducing my background and interests.
- **Modern UI** built with reusable components.
- **Fast Performance** powered by Next.js.

## Technical Details

- **Framework:** Next.js
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** React

## Lessons Learned

Working on this project helped me improve my understanding of component-based architecture, responsive design, project organization, and modern frontend development. It also gave me valuable experience that I applied while building my new portfolio.

## Outcome

While this portfolio has been discontinued, it represents an important step in my journey as a developer. The experience and lessons learned from this project directly influenced the design, structure, and features of my current personal website.
