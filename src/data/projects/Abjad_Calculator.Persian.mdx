# معرفی

محاسبه‌گر ابجد یک ابزار تحت وب است که مقدار عددی کلمات و جملات فارسی و عربی را بر اساس سیستم عددی ابجد محاسبه می‌کند. نتیجه به‌صورت لحظه‌ای نمایش داده می‌شود و رابط کاربری ساده‌ای برای استفاده آسان دارد.

## امکانات

- محاسبه لحظه‌ای مقدار ابجد
- پشتیبانی از حروف فارسی و عربی
- رابط کاربری ساده و سریع
- طراحی واکنش‌گرا

## جزئیات فنی

- React
- JavaScript
- Vite
- Tailwind CSS

## نتیجه

این پروژه نمونه‌ای از پیاده‌سازی منطق پردازش متن در React است و نشان می‌دهد چگونه می‌توان یک ابزار کاربردی و سبک را با فناوری‌های مدرن وب توسعه داد.
