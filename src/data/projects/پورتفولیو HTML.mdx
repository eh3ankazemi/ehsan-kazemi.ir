# معرفی

این پروژه اولین نسخه از وب‌سایت شخصی و نمونه‌کار من است که با استفاده از **HTML، CSS و JavaScript** توسعه داده شد. هدف از ساخت آن، معرفی مهارت‌ها، پروژه‌ها و ایجاد فضایی برای نمایش فعالیت‌های برنامه‌نویسی من بود.

این پروژه نقطه شروع مسیر توسعه وب من بود و تجربه ارزشمندی در زمینه طراحی رابط کاربری، ساخت صفحات واکنش‌گرا و استقرار وب‌سایت روی GitHub Pages برایم به همراه داشت.

## امکانات

- معرفی مهارت‌ها و فناوری‌های مورد استفاده
- نمایش پروژه‌ها و نمونه‌کارها
- طراحی واکنش‌گرا برای موبایل و دسکتاپ
- رابط کاربری ساده و مدرن
- میزبانی رایگان با GitHub Pages

## جزئیات فنی

- **زبان‌ها:** HTML، CSS و JavaScript
- **استایل‌دهی:** CSS
- **میزبانی:** GitHub Pages

## چالش‌ها

یکی از چالش‌های این پروژه، ساخت یک رابط کاربری زیبا و واکنش‌گرا تنها با استفاده از HTML، CSS و JavaScript بود. این تجربه باعث شد درک عمیق‌تری از مفاهیم پایه توسعه فرانت‌اند و طراحی صفحات وب به دست آورم.

## نتیجه

اگرچه بعدها نسخه‌های پیشرفته‌تری از وب‌سایت شخصی خود را با Next.js توسعه دادم، اما این پروژه پایه و آغاز مسیر یادگیری من در توسعه وب بود و همچنان بخش مهمی از روند پیشرفت من را نشان می‌دهد.
