# Introduction

Marks Ket is a cryptocurrency market platform designed to provide users with real-time market information in a clean and responsive interface. The project focuses on performance, accessibility, and a seamless user experience across desktop and mobile devices.

## Features

- **Live Market Data** for cryptocurrencies.
- **Interactive Charts** powered by Chart.js.
- **Search & Filter** cryptocurrencies quickly.
- **Multilingual Support (i18n)** for a wider audience.
- **Progressive Web App (PWA)** support for an app-like experience.
- **Responsive Design** optimized for all screen sizes.

## Technical Details

- **Framework:** Next.js & React
- **Language:** TypeScript
- **Styling:** Tailwind CSS & SCSS
- **Charts:** Chart.js
- **Backend:** Next.js API Routes
- **Database:** MongoDB
- **Localization:** i18n
- **Deployment:** Vercel

## Challenges

One of the biggest challenges was efficiently handling cryptocurrency market data while maintaining fast page loads and a smooth user experience. Caching strategies, optimized API requests, and efficient component rendering helped improve the application's overall performance.

## Outcome

Marks Ket demonstrates my ability to build modern full-stack applications using the latest web technologies. The project combines responsive UI design, backend development, localization, and data visualization into a practical real-world application.
